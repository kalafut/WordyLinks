wordy links

How does it work?

Quickly associate a URL with a set of keywords that you choose. You can later return to that address using those keywords. Keywords can be anything you'll find memorable. Keyword order matters, but spaces and capitalization don't.

Why?

Need to get an unwieldy URL between devices or return to a site using simple, memorable words? Most URL shorteners return cryptic links, and other options usually involve bookmarks, sending yourself emails, etc. Wordy Links requires no accounts, apps or syncing.

Are keyword entries deleted?

No, but after 30 days of disuse (i.e. you've not loaded them), someone else can associate them with a different URL. This ensures that a large pool of short, common words remains available.
